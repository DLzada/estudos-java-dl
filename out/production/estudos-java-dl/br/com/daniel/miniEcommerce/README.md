## 🛒 Mini-Projeto: E-commerce Java (Módulo 2)
> Este projeto implementa um sistema de gerenciamento de produtos utilizando os conceitos fundamentais de POO: classes abstratas, interfaces, herança e polimorfismo.

## Conceitos Aplicados
* **Abstração:** Produto não pode ser instanciado, servindo apenas como modelo.

* **Herança:** Livro e Eletronico herdam atributos de Produto.

* **Interface:** Descontavel separa o comportamento de "ter desconto" da hierarquia principal de produtos.

* **Polimorfismo:** A lista armazena diferentes tipos de produtos, mas os trata de forma genérica até que a verificação específica seja necessária.
